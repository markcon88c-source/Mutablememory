A complete guide to using your development organism

1. What is the Rememger?

The Rememger is a Python-based supervisor organism that:

- launches your creature

- takes full snapshots  
- restores baseline  
- edits daily, meta, and copilot notes  
- opens organs in nano  
- outputs any file or organ like cat  
- snapshots individual organs  
- logs everything to your notes  

It is the memory spine of your entire system.

2. Running the Rememger

Use Python to run it:

`
python3 rememger.py run
`

This will:

- log the launch  
- simulate INIT_COMPLETE  
- simulate HEARTBEAT_START  
- trigger a snapshot

3. Editing Notes

Edit daily note:
`

`

Edit meta note:
`

`

Edit copilot notes:
`

`

These open the files in nano.

To open an organ file in nano:

`

5. Snapshotting an Organ

To snapshot a single organ:

`

- create a new snapshot folder  
- copy the organ file into it  
- log the action to daily/meta/copilot notes
